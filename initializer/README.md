# Vital-Initializer

This project contains the Vital-Initializer.  
It can be used to generate a fully working plugin that contains Vital.  
You can customize your plugin within the generator's setup-steps.

## How to run

To run the initializer, you first need to clone Vital.

### Windows

1. Open a terminal at this project's location (not the root project, this initializer project).
2. Execute `initializer.bat`
3. Follow the on-screen instructions in the terminal.

### Linux / macOS

1. Open a terminal at this project's location (not the root project, this initializer project).
2. Execute `./initializer.sh`
3. Follow the on-screen instructions in the terminal
